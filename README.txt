About Kwartzite
---------------

Kwartzite is a designer-friendly template engine.
Kwartzite is derived from Kwartz.


Features
--------

* Pure HTML temlplate
* Easy to understand
* Not fastest, but enough fast
* Implemented in Python and Ruby
* Java is also supported

See 'python/README.txt' or 'ruby/README.txt' for details.
